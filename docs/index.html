1. Koneksi GitHub

Cara membuat token (sekali saja)

Buka github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Pilih:

  • Repository access: Only select repositories → pilih trustpositif-bot
  • Permissions → Contents: Read and write
  • Permissions → Actions: Read and write

Lalu Generate, salin tokennya, dan paste di atas. Token tidak dikirim ke mana pun selain langsung ke GitHub.
